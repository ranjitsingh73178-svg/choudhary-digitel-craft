# Choudhary Digital Craft

Features:

- E-Book Store
- Technical Classes
- Student Dashboard
- Admin Dashboard
- Course Upload
- E-Book Upload
- Payment System
- PWA Support
- Android APK Support

Installation:

1. Install Python

2. Install Requirements

pip install -r requirements.txt

3. Import database.sql

4. Run Backend

python app.py

5. Open index.html
